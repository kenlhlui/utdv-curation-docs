Features of the U of T Dataverse Curation Tool
===

There are several pages in the U of T Dataverse Curation Tool, each with different functions. Below is a list of the main pages in the tool:

1. [Landing page](./landing.md): the home page of the tool, where you can see the list of your projects, and start a new project, resume an existing project, or delete a project.
2. [New dataset page](./new_dataset.md): the page for starting a new project by providing the dataset information.
3. [Checklist page](./checklist.md): the page for reviewing the checklist items for a project.
4. [Resume project page](./resume.md): the page for resuming an existing project.
5. [Delete project page](./delete.md): the page for deleting an existing project.
